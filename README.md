# 🎧 LDC RADIO

LDC RADIO est une plateforme musicale indépendante pensée pour offrir une expérience simple, fluide et immersive.

---

### ✨ Nouvelle interface
- Visualiseur musical amélioré
- Lecteur audio optimisé

### ⚡ Optimisations
- Meilleures performances sur mobile
- Préchargement des musiques pour réduire au maximum les temps de chargement entre les titres

### 🎵 Contenu
- Nouveaux sons ajoutés quotidiennement

## 🔊 Concept

LDC RADIO, c’est une radio libre.

Une alternative pour écouter de la musique sans contraintes, directement sur le web.

---

## 📱 Compatibilité

- Desktop (site web + app)
- Mobile (site web)
- Tablette (site web)

---

## 💡 Objectif

Proposer une expérience musicale fluide, accessible et en constante évolution.
